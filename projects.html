Tokenization and Bigrams Explore Package

This project is an R package designed to help beginners explore basic text analysis concepts. The package focuses on tokenization and bigrams, which are important foundations for understanding how text can be broken down and analyzed.

The project also connects to the basics of language models, since tokenization and n-grams are key ideas behind how computers process language.

Skills used: R package development, tidytext, tokenization, bigrams, data visualization, documentation
